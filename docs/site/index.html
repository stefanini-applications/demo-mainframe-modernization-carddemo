🏗️ Platform at a Glance

🖥️ Online (CICS)

3270 terminal interface via BMS maps. Each transaction has a COBOL program + BMS screen definition.

⚙️ Batch (JCL)

z/OS JCL jobs for nightly/monthly processing. POSTTRAN, INTCALC, CREASTMT are critical path.

🗄️ VSAM Storage

8 KSDS files with Alternate Indexes. ACCTFILE, CARDFILE, CUSTFILE, TRANFILE are primary master files.

🔒 Security (RACF)

RACF for system-level security. Application-level user types (A/U) stored in USRSEC VSAM file.

📦 Optional: DB2

Transaction type management and authorization audit logging. Demonstrates embedded SQL patterns.

📨 Optional: MQ + IMS

Asynchronous authorization request processing. MQ trigger-and-reply pattern with IMS segment storage.
